# 养殖户头像图片说明

## 目录说明
此目录用于存放养殖户的头像图片。

## 图片命名规则
- 图片文件名：`p1.png`, `p2.png`, `p3.png`, ..., `p10.png`
- 后端会根据养殖户ID自动选择对应的图片：`p{breeder.id % 10 + 1}.png`
- 例如：ID为1的养殖户使用 `p2.png`，ID为10的养殖户使用 `p1.png`

## 图片要求
- **格式**：PNG（推荐）或 JPG
- **尺寸**：建议 200x200 像素或更大（正方形）
- **大小**：建议每个文件不超过 200KB
- **内容**：可以是真实养殖户照片，或使用占位头像图片

## 获取图片的方式

### 方式1：使用占位图片服务（推荐用于开发测试）
可以使用在线占位图片服务生成头像：
- https://ui-avatars.com/ （可生成带名字的头像）
- https://dummyimage.com/ （简单占位图）
- https://picsum.photos/ （随机图片）

### 方式2：使用AI生成头像
- 使用AI工具生成统一的头像风格
- 确保所有头像风格一致

### 方式3：使用真实照片
- 如果有真实养殖户照片，可以处理后放入此目录
- 注意保护隐私，确保有使用授权

### 方式4：使用图标库
- 可以使用统一的图标或头像图标
- 确保风格统一

## 快速开始

### 使用占位图片（最简单）
1. 访问 https://ui-avatars.com/
2. 生成10张不同颜色的头像（可以设置不同的名字或颜色）
3. 下载并重命名为 `p1.png` 到 `p10.png`
4. 放入此目录

### 使用命令行下载（如果安装了curl）
```bash
# 下载10张不同颜色的占位头像
for i in {1..10}; do
  curl -o "p$i.png" "https://ui-avatars.com/api/?name=Farmer$i&size=200&background=random&color=fff&bold=true"
done
```
